# 💍 Web Undangan Pernikahan

Project website undangan pernikahan digital.

## 📌 Deskripsi

Website ini dibuat untuk menampilkan undangan pernikahan secara online dengan tampilan modern dan responsif.

## 🚀 Fitur

- Tampilan responsif
- Animasi interaktif
- Galeri foto
- Countdown acara
- Form RSVP

## 🛠️ Teknologi yang Digunakan

- HTML
- CSS
- JavaScript
- Bootstrap

## 📂 Cara Menjalankan Project

1. Clone repository
2. Buka folder project
3. Jalankan file `index.html` di browser
